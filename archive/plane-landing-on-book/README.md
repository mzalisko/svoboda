# Архів: посадка літака на обкладинку книжки (scroll-driven)

Реалізація ефекту «shared element»: літак з Hero летить по кубічній Безьє
(виліт угору → планерування з нахилом по дотичній → пікірування → flare)
і по скролу сідає точно на місце друкованого літака на обкладинці книжки
в секції «Замовити книгу». Знято з продакшену на користь burst-in анімації
в Hero (літак лишається в Hero), але повністю робоче.

## Вміст

- `initHeroPlaneFly-landing.js` — функція цілком (rAF по скролу, прогрес =
  scrollY до моменту «центр обкладинки в центрі екрана», Безьє з
  тангенціальним поворотом ±16°, посадкові частки на обкладинці).
- `book-cover-plane-erased.png` — обкладинка 1596×2240 (кроп + білий→альфа)
  зі СТЕРТИМ друкованим літаком (щоб спрайт його «заміняв»). Затирання:
  повна заміна зони x 462–1460, y 740–1214 вертикальною інтерполяцією по
  колонках, фазер 18px.

## Як повернути

1. `cp book-cover-plane-erased.png ../../wp-content/themes/svoboda/assets/img/book-cover.png`
2. Вставити функцію в `assets/js/animations.js`, викликати з `init()`.
3. `.section--hero__fly` в CSS повинен мати `transform-origin: 0 0`.
4. Посадкові константи всередині функції: `TARGET = { fx: 0.2995, fw: 0.6053,
   fyc: 0.4362 }` (частки прямокутника <img> обкладинки), `SPRITE_RATIO = 1802/560`.

Контекст: гілка розмови 2026-07-12; спрайти `hero-plane-fly.png` /
`hero-paper.png` лишаються в темі (використовуються і новою анімацією).
